This setup is based on using the 
Multipurpose v2.0 pcb used in Mr Innovative design.
https://oshwlab.com/sharmaz747/multipurpose-pcb

The Nextion (HMI) is setup for a 2.4" display.

The arduino code does not implement limit switch(s) and nothing
added for servo motor use (if you choose to include in your build).
